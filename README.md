# Offer-letter
A clean, responsive HTML and CSS layout for an HR offer letter system. The interface includes a navigation bar, a professional offer letter preview, and action buttons to send the letter or switch between templates. Designed as a foundation for future template management, PDF generation, and email integration.

# Offer Letter Template UI

This project is a clean and professional front-end layout for an HR offer letter system, built using **HTML and CSS**. It provides a structured user interface where HR personnel can preview an offer letter, send it to a successful candidate, or choose a different template.

The design is inspired by real-world corporate HR platforms and focuses on clarity, simplicity, and usability.
